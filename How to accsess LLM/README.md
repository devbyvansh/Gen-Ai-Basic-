# Access Different LLMs

## Overview

This project demonstrates different ways to access Large Language Models (LLMs) for Generative AI applications. It covers local models, cloud-hosted models, managed AI platforms, and multimodal capabilities such as text, image, and audio processing.

## Topics Covered

- Introduction to Large Language Models (LLMs)
- Local vs Cloud LLM Deployment
- Ollama
- Hugging Face Hub
- OpenRouter
- Groq
- Gemini API
- Anthropic (Claude)
- OpenAI Models
- Managed AI Platforms
- AWS Bedrock
- Azure AI Foundry
- Google Vertex AI
- LangChain Integration
- Text-to-Text Models
- Text-to-Image Models
- Image-to-Text Models
- Text-to-Audio Models
- Audio-to-Text Models
- OCR using Vision Models
- Base64 Encoding for Images

## Files

- `access-diff-llm.ipynb` – Implementation and notes for accessing different LLM providers and multimodal models.
- `README.md` – Project documentation.
